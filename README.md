# College Management System - BN231 Assignment 2

This repository contains the Java implementation of a College Management System (CMS) developed for BN231 Assignment 2. The system manages units, unit offerings, instructors, students, and enrolments.

## Features
- Add and view units
- Add and view unit offerings
- Register instructors and assign them to unit offerings
- Register students and enrol them in unit offerings
- View and filter enrolment history by student or unit offering
- Generate student enrolment and class allocation reports
- Store data using ArrayLists
- Save and load data using file I/O
- JUnit test classes for selected system components
- Search and sorting algorithm demonstrations
- Git/GitHub version control evidence

## Technologies Used
- Java
- IntelliJ IDEA
- JUnit 5
- Git and GitHub

## Project Structure
- `src/mainapp` - Main application entry point and algorithm demo classes.
- `src/model` - model classes
- `src/view` - GUI/JFrame classes
- `src/service` - service logic, reports, and file handling
- `src/util` - search and sorting algorithms
- `test` - JUnit test classes
- `data` - saved text file data
